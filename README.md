# QEC AI-Enhanced Decoder

AI-enhanced decoders for quantum error correction (QEC).

## Overview

This repository explores machine-learning-based decoding strategies for quantum error-correcting codes, aiming to improve decoding accuracy, latency, and generalization across noise models compared to classical decoders (MWPM, BP, union-find, etc.).
